Operadores Relacionais

Operam sobre valores relacionais, sendo utilizados em comparação entre valores… 

                        
                            | OPERAÇÃO         | SINAL | EXEMPLO |
                            |   ---            |  ---  |   ---   |
                            | Atribuição       |   =   | x = 10  |
                            | Comparação       |   ==  | x == 10 |
                            | Diferente        |   !=  | x != 10 |
                            | Maior que        |   >   | x > 10  |
                            | Menor que        |   <   | x < 10  |
                            | Maior ou igual a |   >=  | x >= 10 |
                            | Menor ou igual a |   <=  | x <= 10 |

Operadores Lógicos

Operam sobre valores booleanos, comparando condições lógicas. 

                        
                            | OPERAÇÃO | SINAL | EXEMPLO             |
                            | ---      | ---   | ---                 |
                            | Negação  |   !   | !true == false      |
                            |                  |  !false == true     |
                            | E/ And   |   &&  | (x < 10) && (x > 0) |
                            | Ou/ Or   |   ||  | (x < 10) || (x > 0) |

Para entender melhor sobre operadores lógicos, é interessante consultar o que chamamos de “Tabela Verdade”, envolvendo o uso dos operadores && e ||. 

                        
                            | P | Q | P && Q |                                          | P | Q | P || Q |
                            |  ---  |  ---   |                                          | ----  | ------ | 
                            | V | V | V      |                                          | V | V | V      |
                            | V | F | F      |                                          | V | F | V      |
                            | F | V | F      |                                          | F | V | V      |
                            | F | F | F      |                                          | F | F | F      |                   
                            
                && → Será True se TODOS os operandos forem True.               || → Será True se no mínimo UM dos operandos for True.

Estrutura de Decisão

A estrutura condicional permite que o programa execute diferentes ações com base em condições específicas. Assim:

- Se a condição for atendida (T), o programa segue um caminho;

- Se a condição não for atendida (F), o programa segue outro caminho.

Diante dessas circunstâncias, é possível destacar as principais estruturas de decisão:

IF

Irá realizar algo caso a sua condição retorne true. 

            
                1       int a = 10;
                2       
                3       void setup() {
                4         Serial.begin(9600);
                5       }
                6       
                7       void loop() {
                8         if (a == 10) {
                9           Serial.println("A condição e verdadeira!");
                10         }
                11       delay(1000);
                12     }

ELSE IF

Múltiplas condições em forma de cascata, o else if adiciona mais um condição caso o if anterior retorne false, permitindo utilizar um else no final para exceções. 

            
             1       int a = 10;
             2       
             3       void setup() {
             4         Serial.begin(9600);
             5       }
             6      void loop() {
             7       if (a < 10) {
             8        Serial.println("a e menor do que 10");
             9       } else if (a > 10) {
             10       Serial.println("a e maior do que 10");
             11      } else if (a == 10) {
             12       Serial.println("a e igual a 10");
             13      } else {
             14       Serial.println("a e um valor inválido!");
             15      }
             16       delay(1000);
             17      }

Ao rodar o exemplo, será obtido o terceiro print, “A é igual a 10”, o código irá rodar as condições uma a uma, porém como a variável a não é maior nem menor do que 10, elas retornaram false, enquanto a condição de igual retornará true e executará o código em seu interior.

Colocando em Prática

  1. Incrementar o valor de uma variável ao apertar um botão e mostrar esse valor na tela (Serial monitor).
  2. Incrementar o valor de uma variável ao apertar um botão e decrementar a mesma variável ao apertar outro botão, mostrar os valores na tela.
  3. Ligar LED com um botão e desligar o mesmo LED com outro botão.
  4. Apertar um botão para ligar um LED com 3 intensidades de iluminação, um segundo botão para desligar o LED.
  5. Semáforo - fazer 3 LED's piscarem de forma intercalada caso um botão for precionado.
