Atividades Aula 2

  • Fazer um LED piscar com um intevalos de tempo.
  • Semáforo - fazer 3 LED's picarem de forma intercalada.
  • Variar iluminação do LED com potenciômetro.
  • Variar iluminação dos LED's de forma inversa com potenciômetro.

Atividades Aula 3

  • Incrementar o valor de uma variável ao apertar um botão e mostrar esse valor na tela (Serial monitor).
  • Incrementar o valor de uma variável ao apertar um botão e decrementar a mesma variável ao apertar outro botão, mostrar os valores na tela.
  • Ligar LED com um botão e desligar o mesmo LED com outro botão.
  • Apertar um botão para ligar um LED com 3 intensidades de iluminação, um segundo botão para desligar o LED.
  • Semáforo - fazer 3 LED's piscarem de forma intercalada caso um botão for precionado.

Projeto

Para a finalização deste curso vamos tentar colocar em prática o que aprendemos nas aulas passadas. Pense em um projeto, uma ideia de algo que pode ser feito usando o Arduino, automatize algo ou replique um projeto simples.

Requisitos para a realização da atividade:

  • Faça uma breve descrição do funcionamento do projeto.
  • Liste os componentes que serão utilizados.
  • Tente usar a maior parte que conseguir das funções que aprendeu.
  • Converse com os tutores sobre a ideia.
