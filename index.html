Variáveis

Uma variável é um símbolo que representa um valor desconhecido ou arbitrário em um contexto específico. Ela pode assumir diferentes valores ao longo de um programa ou cálculo. As variáveis são usadas para armazenar e manipular dados em algoritmos e programas de computador. Por exemplo, em um programa simples, podemos ter a variável "idade" que armazena a idade de uma pessoa. O valor dessa variável pode ser alterado ao longo do programa, à medida que a idade da pessoa muda.

Tipos de Dados

Em boa parte das linguagens de programação, é necessário informar o tipo de dado que será armazenado dentro da variável. Assim, cada tipo armazena um formato e carrega suas peculiaridades, sendo os principais:

                        | ---    |  --- | --- | ---  | -|                |                         |
                        | int    | Números inteiros     | %i             | int x                   |
                        | float  | Números decimais     | %f             | float x                 |
                        | char   | 1 Caractere          | %c             | char x;                 |
                        | string | Cadeia de caracteres | %s             | char[] = “Waldir Braz”; |
                                                                           char x[3] = {’Y’, ‘N’}; |

Estrutura de Decisão

A estrutura condicional permite que o programa execute diferentes ações com base em condições específicas. Assim:

- Se a condição for atendida (T), o programa segue um caminho;

- Se a condição não for atendida (F), o programa segue outro caminho.

Diante dessas circunstâncias, é possível destacar as principais estruturas de decisão:

IF

Irá realizar algo caso a sua condição retorne true. 

                        
                     1       int a = 10;
                     2       
                     3       void setup() {
                     4         Serial.begin(9600);
                     5       }
                     6       
                     7       void loop() {
                     8         if (a == 10) {
                     9           Serial.println("A condição é verdadeira!");
                     10         }
                     11       delay(1000);
                     12     }

ELSE IF

Múltiplas condições em forma de cascata, o else if adiciona mais um condição caso o if anterior retorne false, permitindo utilizar um else no final para exceções. 

                        
                     1       int a = 10;
                     2       
                     3       void setup() {
                     4         Serial.begin(9600);
                     5       }
                     6      void loop() {
                     7       if (a < 10) {
                     8        Serial.println("a é menor do que 10");
                     9       } else if (a > 10) {
                     10       Serial.println("a é maior do que 10");
                     11      } else if (a == 10) {
                     12       Serial.println("a é igual a 10");
                     13      } else {
                     14       Serial.println("a é um valor inválido!");
                     15      }
                     16       delay(1000);
                     17      }

Ao rodar o exemplo, será obtido o terceiro print, “A é igual a 10”, o código irá rodar as condições uma a uma, porém como a variável a não é maior nem menor do que 10, elas retornaram false, enquanto a condição de igual retornará true e executará o código em seu interior.

Switch-Case

Estrutura de escolha ou seleção utilizada para selecionar uma opção verdadeira, isto é, quando é preciso uma quantidade maior de condições. Além disso, uma de suas exclusividades é aceitar critérios apenas do tipo inteiro ou char.

A instrução break associada a cada case garante que o programa saia da condicional switch e execute a instrução que segue logo após a switch. Caso o break seja omitido, o programa continua a execução para a próxima instrução dentro do switch (até encontrar um “break”) - se nenhum “break” for encontrado, o programa pode executar códigos indesejados ou até entrar em um loop infinito. 

                    
                     1   int input;
                     2
                     3  void setup() {
                     4    Serial.begin(9600);
                     5  }
                     6   
                     7  void loop() {
                     8    switch(input){
                     9      case 1: 
                     10       Serial.println("Opção 1.");
                     11     case 2: 
                     12       Serial.println("Opção 2.");
                     13     case 3: 
                     14       Serial.println("Opção 3.");
                     15     default:
                     16       Serial.println("Opção reserva.");
                     17   }
                     18   delay(1000);
                     19 }

Estruturas de Repetição

Repete um trecho de código até que seja indicado o contrário, isso pode ser atingido com um contador ou uma condição, os tipos de estrutura de repetição são for, while, e do while.

For

For é uma estrutura de repetição que só funciona em loops controlados por contador, sendo necessário o uso de uma variável como contador para sofrer um acréscimo (x++) ou decréscimo (x--) ao final do bloco de instruções interno.

Sua lógica e estrutura é composta da seguinte forma: 

                    
1  for(inicialização; condição; contador/incremento){       | 1  int i;
2   codificação (True);                                     | 2    for(int i = 0; i >= 10; i++) {
3  }                                                        | 3    Serial.println("Valor de i:");
4                                                           | 4     Serial.println(i);
5                                                           | 5	}
 - Inicialização

   Ao contador é atruibuído um valor inicial                 Aqui o i será impresso 11 vezes, 
                                                             seu valor aumentará a cada repetição do laço, indo de 0 a 11.

Incremento

Após a execução do bloco interno, o contador usa formas contraídas para reiniciar o loop.

for(inicialiação; condição; i++){

While

Dada uma verificação/condição, o bloco de instruções é executado enquanto a condição resultar True e, quando resultar False, a execução sai do laço de repetição e executa os próximos comandos. A condição while deve ser alterada dentro do while, com um contador. Assim, essa estrutura faz a verificação das condições e executa as instruções dentro de seu laço. 

                    
                        int a = 1;

                        while(a) {
                            Serial.println("rodando");
                        }

                        Código rodará “infinitamente”

Do while

Essa estrutura de repetição repete a instrução enquanto a condição for True, mas a condição é verificada após a instrução “do” (ao contrário da estrutura while). Portanto, se o contador for alterado dentro da instrução, isso afetará a verificação da condição. Esta estrutura é bastante utilizada quando é preciso executar o bloco de instruções interno pelo menos uma vez.

Vetor

Em programação, um vetor é uma estrutura de dados que armazena uma sequência de elementos do mesmo tipo em uma única variável. Cada elemento do vetor é acessado por um índice, que representa sua posição na sequência. Os vetores são usados para armazenar coleções de dados de forma organizada e eficiente.

Em C, um vetor é declarado especificando o tipo dos elementos e o tamanho do vetor. Aqui está um exemplo de declaração de um vetor de inteiros em C: 

                    
                     1   void setup() {
                     2  
                     3   Serial.begin(9600); 
                     4  }
                     5  
                     6   
                     7  void loop() {
                     8    int numeros[5]; 
                     9     numeros[0] = 10;
                     10    numeros[1] = 20;
                     11    numeros[2] = 30;
                     12    numeros[3] = 40;
                     13    numeros[4] = 50;
                     14         
                     15    Serial.println("Elementos do vetor:");
                     16    Serial.println(numeros[0]);
                     17    Serial.println(numeros[1]);  
                     18    Serial.println(numeros[2]);
                     19    Serial.println(numeros[3]);
                     20    Serial.println(numeros[4]);
                     21   delay(1000);
                     22 }

Neste exemplo, declaramos um vetor de inteiros chamado numeros com tamanho 5. Cada elemento do vetor é acessado utilizando o nome do vetor seguido de um índice entre colchetes, como numeros[0], numeros[1], e assim por diante.

Funções

Em programação, uma função é um bloco de código que realiza uma tarefa específica e pode ser chamado ou invocado a partir de outros locais do programa. Uma função pode receber valores de entrada, chamados de argumentos ou parâmetros, e pode retornar um valor como resultado.

Aqui está um exemplo simples de uma função em C que recebe dois números inteiros e retorna a soma deles: 

                    
                        1  void setup() {
                        2    Serial.begin(9600); 
                        3  }
                        4  
                        5  int somar(int num1, int num2) {
                        6    int resultado = num1 + num2;
                        7    return resultado;
                        8  }
                        9  
                        10   void loop() {
                        11    int a = 5;
                        12    int b = 3;
                        13    int resultado_soma = somar(a, b);
                        14  
                        15      Serial.print("O resultado da soma de ");
                        16      Serial.print(a);
                        17      Serial.print(" e ");
                        18      Serial.print(b);
                        19      Serial.print(" é: ");
                        20      Serial.println(resultado_soma);
                        21  
                        22    delay(1000);
                        23  }

Essa é apenas uma ilustração básica de uma função em programação. As funções podem ser mais complexas, aceitar vários parâmetros, não retornar valores ou ter outros comportamentos específicos, dependendo da linguagem de programação utilizada.

As funções desempenham um papel crucial na programação, independentemente da linguagem utilizada, incluindo a programação para Arduino. Elas permitem que você estruture seu código em segmentos independentes para realizar tarefas específicas. A criação de funções é especialmente útil quando você precisa executar uma mesma ação várias vezes em um programa.

Aqui estão algumas das principais razões pelas quais o uso de funções é importante:

1. Organização: As funções ajudam os programadores a se manterem organizados. Ao dividir o código em funções, é possível conceituar o programa de forma mais clara, facilitando a compreensão e a manutenção futura.

2. Reutilização de código: Ao definir uma função para uma determinada tarefa, você pode reutilizá-la em diferentes partes do programa. Isso torna o código mais compacto e reduz a redundância, pois as seções do código são reaproveitadas várias vezes.

3. Modularidade: O uso de funções torna o código mais modular, permitindo que você separe diferentes partes lógicas do programa em funções independentes. Isso facilita a compreensão do código e sua reutilização em outros projetos.

4. Facilidade de depuração: Ao escrever código em forma de funções, você pode isolar e depurar cada função separadamente. Isso torna a identificação e correção de erros mais fácil, uma vez que você só precisa analisar e depurar uma função de cada vez.

No contexto específico do Arduino, a utilização de funções é fundamental. Além das funções padrão setup() e loop() que são necessárias em todo esboço do Arduino, é possível criar funções personalizadas para realizar tarefas específicas. Essas funções podem ser declaradas fora dos colchetes das funções setup() e loop(), seguindo a sintaxe padrão da linguagem C++.

Ao criar funções no Arduino, você pode encapsular diferentes partes do código em funções independentes, facilitando a organização e a compreensão do programa. Por exemplo, você pode criar uma função para ler dados de sensores, outra para processar esses dados e uma terceira para controlar ações com base nos resultados. Isso permite um código mais estruturado e modular, além de facilitar a manutenção e a reutilização em outros projetos Arduino.
